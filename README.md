# IND 290 Food Delivery Matching Project

## Overview
This project builds machine learning models to optimize food delivery dispatching, including ETA prediction, order acceptance probability, and system-level queueing optimization.

## Data Source
The dataset is obtained from: https://github.com/meituan/Meituan-INFORMS-TSL-Research-Challenge

## Models
- ETA Prediction Model: predicts delivery time
- Acceptance Model: predicts order acceptance probability
- Queueing System: models system congestion and stability

## Tech Stack
Python, LightGBM, Pandas, NumPy, Scikit-learn

## Structure
Code: model training and system logic  
Models: trained .pkl files
